Add a new task

The following command will add a new event, deadline or task based on the input parameters.

  1. For event task
    add <title> -[startDate] -[startTime] -[endDate] -[endTime]
  2. For deadline task
    add <title> -[date] -[time]
  3. For floating task
    add <title>

Parameter

  1. [startDate] / [endDate]
    You can specify the date based on the format DD/MM/YYYY or a natural day such as monday, tuesday, next week, next monday.
  2. [startTime] / [endTime]
    Time will be in the following format HH:MM.

Edit your task

The following command will edit your task based on the input parameters.

  1. For event task
    edit <Task ID> -[newTitle] -[newStartDate] -[newStartTime] -[newEndDate] -[newEndTime]
  2. For deadline task
    edit <Task ID> -[newTitle] -[newEndDate] -[newEndTime]
  3. For floating task
    edit <Task ID> -[newTitle]

Parameter

  1. <Task ID>
    ID is the number that is assigned to the tasl. It can be found at the left of the task title in the list.
  2. [newTitle]
    If a new title is not specified in the command, system will assume the title remains the same.
    Specify a new title will replace the old title of the task.
  3. [newStartDate] / [newEndDate]
    You can specify the date based on the format DD/MM/YYYY or a natural day such as monday, tuesday, next week, next monday.
  4. [newStartTime] / [newEndTime]
    Time will be in the following format HH:MM.

Delete your task

This command will delete the task that you specify. More detail is below.

  1. To delete a specific task
    delete <Task ID>
  2. To delete all tasks
    delete a

Parameter

  1. <Task ID>
    ID is the number that is assigned to the task. It can be found at the left of the task title in the list.
  2. a
    Delete all tasks that are stored in Pista.

Mark done & undone

This command will mark your task as done or undone.

  1. To mark a task as complete
    mark <Task ID> -done
  2. To mark a task as not complete
    mark <Task ID> -undone

Parameter

  1. <Task ID>
    ID is the number that is assigned to the task. It can be found at the left of the task title in the list.

Remind yourself

The following command will set or unset an alarm for a specific task.

  1. To set an alarm
    remind <Task ID> -[date] -[time]
  2. To unset an alarm
    mark <Task ID> -off

Parameter

  1. <Task ID>
    ID is the number that is assigned to the task. It can be found at the left of the task title in the list.
  2. [date]
    You can specify the date based on the format DD/MM/YYYY or a natural day such as monday, tuesday, next week, next monday.
  3. [Time]
    Time will be in the following format HH:MM.
  4. off
    It will specify that the alarm will be off.
